## web3 go 入门之路

### 一、了解密码学、区块链、比特币（Bitcoin）、以太坊、智能合约

密码学： 非对称加密、私钥公钥（公钥加密、私钥解密；私钥签名、公钥验签）

区块链：通过hash连接每一个区块；当前区块拥有前一个区块的hash；每一个区块有块头和块体

存放交易的结构体：merkel tree

比特币：Bitcoin  BTC

以太坊：基于账户的管理方式
    内部有：状态树、交易树、收据树   数据结构 -> MPT

智能合约 solidity
